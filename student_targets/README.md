# student_targets

This directory contains starter templates for Part 2 of Project 4.

Files:
- `target1_property_fuzzer.cc`
- `target2_property_fuzzer.cc`

These are starter libFuzzer harnesses only. They are intentionally incomplete:
- the example logic is not meant to be submitted unchanged
- the assertions are commented out
- students should replace the logic and property with their own design
